Политика конфиденциальности

Последнее обновление: 4 мая 2026 г.

1. Кто мы

Azanov Travel — частная туристическая компания, зарегистрированная в Пхукете, Таиланд. Мы специализируемся на организации индивидуальных туров, аренде яхт и вилл.

Контакты: info@azanovtravel.com · +66 6-3541-2949

2. Какие данные мы собираем

При заполнении формы заявки на сайте мы собираем:

  • Имя
  • Номер телефона / WhatsApp
  • Email (необязательно)
  • Предпочтения по туру (тип опыта, даты, бюджет, количество гостей)
  • Пожелания в свободной форме

Мы также автоматически собираем:

  • Данные аналитики (Google Analytics) — анонимная статистика посещений
  • IP-адрес — для защиты от спама

3. Как мы используем данные

Ваши персональные данные используются исключительно для:

  • Формирования и отправки персонального предложения
  • Связи с вами через выбранный канал (WhatsApp, Telegram, звонок)
  • Управления заявками в CRM-системе (AmoCRM)
  • Отправки уведомлений о вашей заявке через Telegram-бот

Мы не продаём и не передаём ваши данные третьим лицам для маркетинговых целей.

4. Обработчики данных

Для обработки заявок мы используем следующие сервисы:

  • AmoCRM — CRM-система для управления заявками (хранение, статус)
  • Telegram Bot API — мгновенное уведомление нашей команды
  • Google Analytics — анонимная веб-аналитика

5. Хранение данных

Персональные данные хранятся в CRM-системе до момента:

  • Завершения бронирования и оказания услуг, или
  • Вашего запроса на удаление данных

Мы не храним данные банковских карт. Все платежи обрабатываются напрямую при встрече или через защищённые банковские каналы.

6. Ваши права

В соответствии с Законом Таиланда о защите персональных данных (PDPA) вы имеете право:

  • Запросить информацию о хранящихся данных
  • Потребовать исправления неточных данных
  • Потребовать удаления ваших данных
  • Отозвать согласие на обработку данных

Для реализации этих прав свяжитесь с нами: info@azanovtravel.com

7. Файлы cookie

Наш сайт использует:

  • Google Analytics cookies — для анализа посещаемости (анонимно)
  • Языковые предпочтения — localStorage для запоминания выбора языка (RU/EN)

Мы не используем рекламные или трекинговые cookie третьих сторон.

8. Безопасность

Мы применяем стандартные меры защиты:

  • SSL-шифрование (HTTPS) для всех передач данных
  • Ограничение частоты запросов к API (Rate Limiting)
  • Ограниченный доступ к CRM только для сотрудников компании
